BioScript Documentation

Overview

BioScript is a Visual Studio Code extension designed to assist with bioinformatics analyses. It provides a command-driven interface for manipulating and analyzing DNA and RNA sequences, supporting operations such as sequence loading, transcription, translation, mutation simulation, and more.

Installation

To install BioScript:

  1. Place the extension files into your Visual Studio Code extensions directory.
  2. Open Visual Studio Code.
  3. Run the Activate Extension command via the command palette (Ctrl+Shift+P or Cmd+Shift+P on macOS).

Features

  • Load and Manage Sequences: Load DNA or RNA sequences from files and view sequence information.
  • Manipulate Sequences: Reverse, complement, transcribe, and translate sequences.
  • Simulate Genetic Variations: Introduce single nucleotide polymorphisms (SNPs), mutations, and trim sequences.
  • Analyze Sequences: Calculate nucleotide composition, align sequences, and compute similarity scores.
  • Annotate Sequences: Add annotations to specific positions within sequences.
  • Pattern Matching: Filter sequences by searching for specific patterns.
  • Random Sequence Generation: Generate random DNA sequences of a specified length.
  • Domain Prediction: Predict functional domains within sequences based on known patterns.

Usage

BioScript is command-driven and integrates with Visual Studio Code's command palette. Follow these steps to execute commands:

  1. Open a new text document in Visual Studio Code.
  2. Type the desired commands (one per line) into the document.
  3. Press Ctrl+Shift+P (or Cmd+Shift+P on macOS) to open the command palette.
  4. Type Evaluate Modela and select the command to run the BioScript code in the document.
  5. Results and outputs will appear as notifications within Visual Studio Code.

Commands

BioScript supports a variety of commands to manipulate and analyze sequences. Below are some of the available commands:

Load Sequence

load sequence from "filename"

Loads a DNA or RNA sequence from a specified file. The file should be in the same directory as the workspace.

Show Sequence Information

show info

Displays the header and the first 200 bases of the currently loaded sequence.

Other Commands

  • show composition - Displays nucleotide composition.
  • reverse sequence - Reverses the sequence.
  • complement DNA - Generates the complement of the DNA sequence.
  • transcribe DNA to RNA - Transcribes DNA into RNA.
  • translate RNA to protein - Translates RNA into protein.
  • mutate at position to "" - Introduces a mutation at the specified position.
  • generate random sequence of length - Generates a random DNA sequence.

Helper Functions

BioScript uses several helper functions to support its operations, such as complementDNA(sequence), translateRNAtoProtein(rnaSequence), and calculateComposition(sequence).

Example Workflows

Workflow 1: Load and Analyze a Sequence

load sequence from "human_gene.fasta"
show info
show composition
transcribe DNA to RNA
translate RNA to protein

Workflow 2: Simulate Mutations and Annotate

load sequence from "sample_dna.fasta"
mutate at position 250 to "G"
introduce SNP at 100,300,500
annotate with "SNP Location" 100
show annotations

Workflow 3: Filter and Align Sequences

load sequence from "genome_segment.fasta"
filter sequence with pattern "TATAAA"
align with "ATGCGTAGCT"
